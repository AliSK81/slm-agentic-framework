# Security Policy

## Supported Versions

This project is an MSc thesis research implementation. Only the latest version on the `master` branch receives security attention.

| Version | Supported |
|---------|-----------|
| latest (master) | ✅ |
| older releases  | ❌ |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub Issues.**

If you discover a security vulnerability, please send an email to **aliebrahimi6739@gmail.com** with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact

You can expect an acknowledgement within **72 hours** and a status update within **7 days**.

## Secrets & API Keys

Never commit API keys or secrets to this repository. Use `.env` (gitignored) and refer to `.env.example` for the required environment variables.
